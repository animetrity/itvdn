· Що таке Серіалізація/десеріалізація?
---> Це процес перетворення Java об'єкту у байт код для збереження його стану, а також його читання з байт коду

· Що таке Маршаллінг/демаршаллінг?
---> Це також схожий процес до серіалізації, проте записує також дані про сам об'єк для відновлення його структури, тобто парсеру не обов'язково завчасно знати структуру об'єкту щоб отримати дані

· Розкажіть про XML, у чому різниця між HTML?
---> html це мова розмітки, тобто вона відображає дані, в той час як xml використовується для зберігання і передачі ніформації.

· Які парсери в XML знаєте?
---> SAXParser, JAXB, XMLParserDOM

· Що робить JSON?
---> це стандарт передачі даних, зачасту використовується в вебсервісах

· Що таке JAXB?
---> Парсер XML
