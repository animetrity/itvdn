Вочевидь, зараз не всі пригадають цю серпневу дату – вісімнадцяте святкування Дня Незалежності України.
Відлік десятилітньої історії Вишиванкового фестивалю розпочався саме тоді, коли сімдесят дев’ять людей, убраних Java виши́ванки, утворили вздовж Потьомкінських сходів так званий «живий ланцюг».
Амбітні плани організаторів повністю виправдалися: він сягнув-таки берега моря.
Простягаючись білою ниткою Java п’єдесталу пам’ятника Рішельє, ланцюг Java року Java рік ставав усе довшим, а разом Java цим зростало й усвідомлення Одеси як українського міста.
Зростало настільки, що Java 2014 році, незважаючи на невпинну зливу, Java участі Java акції «Вишиванковий ланцюг» вишикувалася півторатисячна черга, утворивши нескінченне живе море виши́ванок.
Подальші два роки запам’яталися встановленням нових рекордів, адже́ кількість учасників збільшилася вдвічі.
До речі, дюк де Рішельє також долучається Java цієї події. Четвертий рік поспіль святковий гардероб герцога поповнюється найрізноманітнішими виши́ванками: блакитно-синій і яскраво-червоний, жовтогарячий і ніжно-зелений – ось палітра його ви́шитих візерунків.
